# Re:flect — Інформаційна система інтерактивної презентації айдентики бренду

Веб-застосунок для управління та інтерактивного відображення айдентики бренду Re:flect.

## Що це таке

Система складається з двох частин:

- **Публічна презентація** (`/`) — інтерактивний сайт з айдентикою бренду: логотип, кольори, шрифти, мокапи, слогани. Всі дані підвантажуються з сервера динамічно.
- **Адмін-панель** (`/admin.html`) — закрита сторінка для управління контентом бренду: додавання та редагування кольорів, мокапів, слоганів, основної інформації.

## Технічний стек

| Частина | Технологія |
|---|---|
| Backend | Node.js + Express |
| База даних | SQLite (better-sqlite3) |
| Frontend | HTML + CSS + Vanilla JS |
| Завантаження файлів | Multer |

## Структура проєкту

```
reflect-site/
├── backend/
│   ├── server.js         — головний файл сервера
│   ├── db.js             — підключення до БД та початкові дані
│   └── routes/
│       ├── brand.js      — маршрути для інфо бренду
│       ├── colors.js     — маршрути для кольорів
│       ├── mockups.js    — маршрути для мокапів
│       └── slogans.js    — маршрути для слоганів
└── frontend/
    ├── index.html        — публічна презентація
    ├── admin.html        — адмін-панель
    └── images/           — зображення мокапів та логотипу
```

## Запуск локально

### 1. Встановити залежності

```bash
cd backend
npm install
```

### 2. Запустити сервер

```bash
node server.js
```

### 3. Відкрити в браузері

- Публічна частина: [http://localhost:3000](http://localhost:3000)
- Адмін-панель: [http://localhost:3000/admin.html](http://localhost:3000/admin.html)

## API ендпоінти

| Метод | URL | Опис |
|---|---|---|
| GET | `/api/brand` | Всі дані бренду одним запитом |
| PUT | `/api/brand` | Оновити інфо бренду |
| GET | `/api/colors` | Список кольорів |
| POST | `/api/colors` | Додати колір |
| PUT | `/api/colors/:id` | Оновити колір |
| DELETE | `/api/colors/:id` | Видалити колір |
| GET | `/api/mockups` | Список мокапів |
| POST | `/api/mockups` | Завантажити мокап (multipart) |
| DELETE | `/api/mockups/:id` | Видалити мокап |
| GET | `/api/slogans` | Список слоганів |
| POST | `/api/slogans` | Додати слоган |
| PUT | `/api/slogans/:id` | Оновити слоган |
| DELETE | `/api/slogans/:id` | Видалити слоган |
